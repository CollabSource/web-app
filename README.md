# README

### starting the app
- backend: `$ go run back-end/main.go`
    - this will install necessary dependencies
- frontend: `$ npm start`
### tests (back end)
- `$ cd tests/testName && go test`
    - for contract tests, start server first
## dependencies 
### backend (golang)
- download the dmg from https://go.dev/doc/install
### frontend (ts)
- `$ npm install`
